#  Stock Portfolio Tracker (Python)

This is a simple and beginner-friendly **Stock Portfolio Tracker** made in Python.  
You enter stock symbols and the number of shares you own, and the program calculates your **total investment value**.  
It also saves the final summary in `portfolio.txt`.


##  Features

-  Accepts stock symbols interactively
-  Calculates value based on share quantity
-  Warns if an invalid stock symbol is entered
-  Saves total investment to a file
-  Great project to learn dictionaries, loops, and file handling
